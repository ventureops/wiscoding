# wiscoding

> 바이브코딩과 Claude 활용 노하우를 공유하는 작은 자료실

10명 규모의 커뮤니티에서 함께 쌓아가는 실전 자료 모음입니다. 누구나 이슈로 팁을 제출할 수 있고, 운영자 검토를 거쳐 정식 문서로 편입됩니다.

## 어떻게 참여하나요

세 가지 방법이 있습니다. 부담 없는 쪽부터 시작하세요.

1. **이슈로 팁 제출** — [새 이슈](../../issues/new/choose) 에서 "새로운 팁" 템플릿을 골라 작성합니다. 운영자가 검토 후 정식 문서로 편입합니다.
2. **Discussions에서 질문/토론** — 가벼운 질문이나 자유로운 토론은 [Discussions](../../discussions) 에서 합니다.
3. **Pull Request** — 기존 문서의 오타·내용 수정은 직접 PR을 보내주세요. 자세한 규칙은 [CONTRIBUTING.md](CONTRIBUTING.md) 참고.

## 운영 원칙

- **검증된 것만 올린다** — 본인이 직접 써보고 동작을 확인한 내용을 공유합니다.
- **출처를 밝힌다** — 다른 자료를 참고했다면 링크와 함께 표기합니다.
- **민감정보는 금지** — API 키, 토큰, 사내 코드, 개인 식별 정보는 절대 포함하지 않습니다.
- **친절하게 쓴다** — 처음 보는 사람도 따라할 수 있게 단계별로 적습니다.
- **짧고 명확하게** — 한 문서는 하나의 주제만 다룹니다.
