# 도구·통합 (Tools)

함께 쓰면 좋은 **외부 도구**, **통합 사례**, **자동화 스크립트**에 관한 자료를 모읍니다.

Claude Code 자체 기능은 [`docs/claude-code/`](../claude-code/), AI 협업 일반론은
[`docs/vibe-coding/`](../vibe-coding/) 으로 가주세요. 여기서는 **다른 도구**를 어떻게 엮어 쓰는지를 다룹니다.

## 다루는 주제

- **에디터·IDE** — VSCode 확장, JetBrains 플러그인, Vim/Neovim 통합
- **버전 관리** — Git 고급 사용, GitHub CLI(`gh`), GitLab, worktree
- **CLI 도구** — `jq`, `fzf`, `ripgrep`, `bat`, `delta` 등과 AI 워크플로 연계
- **자동화** — 셸 스크립트, Makefile, just, GitHub Actions 워크플로
- **MCP 서버** — 다양한 MCP 서버 연결 사례 (Notion, Slack, Linear, Google Drive 등)
- **AI 보조 도구** — Cursor, Aider, Cline, Continue 등 다른 AI 도구 비교·사용기
- **협업 도구** — Linear, Notion, Slack, Discord 자동화·연동
- **배포 플랫폼** — Vercel, Cloudflare, Fly.io, Railway 등 셋업·팁
- **DB·데이터** — 로컬 DB 셋업, 시드 스크립트, 마이그레이션 도구
