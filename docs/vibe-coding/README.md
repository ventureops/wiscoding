# 바이브코딩 (Vibe Coding)

AI와 함께 코딩하는 **워크플로**, **프롬프트 패턴**, **협업 방식**에 관한 자료를 모읍니다.

특정 도구(Claude Code, Cursor, Copilot 등)에 종속되지 않은, AI 협업 자체의 노하우를 다룹니다.
도구별 구체 사용법은 [`docs/claude-code/`](../claude-code/) 또는 [`docs/tools/`](../tools/) 로 가주세요.

## 다루는 주제

- **워크플로 패턴** — 기획→구현→검증을 AI와 어떻게 분담할지, 세션 단위로 나누는 방법
- **프롬프트 패턴** — 자주 쓰는 프롬프트 구조, 효과적이었던 표현, 안티패턴
- **컨텍스트 관리** — 어떤 정보를 먼저 주고, 어떤 정보를 미루며, 언제 새 세션을 열지
- **검증 루틴** — AI가 만든 코드를 어떻게 신뢰성 있게 확인하는지
- **TDD·디버깅과 AI** — 테스트 기반 개발과 AI 협업, AI와 함께 디버깅하는 절차
- **계획·브레인스토밍** — 큰 작업 시작 전 AI와 함께 설계 다듬기
- **회고·메타** — "이 작업 AI에 맡겼으면 더 빨랐을까" 같은 회고 기록
- **함정 사례** — AI 추천을 그대로 받아 생긴 사고, 학습 포인트
