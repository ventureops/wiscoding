# 템플릿 (Templates)

복사해서 바로 쓸 수 있는 **재사용 가능한 자료**를 모읍니다.

## 어떤 템플릿이 들어오나요

- **프롬프트 템플릿** — 자주 쓰는 작업 (코드 리뷰, 리팩터링, 디버깅 등)을 위한 프롬프트
- **문서 템플릿** — README, ADR, PR 설명, 회고 등의 골격
- **설정 파일 템플릿** — `.claude/settings.json`, `.editorconfig`, `.gitignore` 모음
- **이슈·PR 템플릿** — 다른 프로젝트에서도 쓸 수 있는 GitHub 템플릿
- **워크플로 템플릿** — GitHub Actions, 배포 스크립트
- **체크리스트** — 배포 전, 릴리스 전, 회고 전 등 점검용

## 사용 방법

각 템플릿 파일 상단에 다음이 적혀 있습니다.

- **용도** — 어떤 상황에서 쓰는지
- **사용법** — 어떻게 복사해서 채우는지
- **출처** — 어디서 영감을 받았거나, 누가 만들었는지
- **변형 포인트** — 어떤 부분을 자기 프로젝트에 맞게 바꿔야 하는지

## 자료 추가하기

이슈 템플릿 **"새로운 팁"** 에서 분류를 `템플릿`으로 골라 제출하시거나,
직접 PR로 이 폴더에 파일을 추가해주세요.

파일명 예시:
- `prompt-code-review.md`
- `adr-template.md`
- `gitignore-node-claude.txt`

자세한 작성 규칙은 루트의 [CONTRIBUTING.md](../CONTRIBUTING.md) 참고.
